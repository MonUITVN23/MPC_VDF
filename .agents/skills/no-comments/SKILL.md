# Quy định Tắt Comment (No Comments Rule)

1. **Không tạo comment mới**: Các code triển khai từ nay về sau **KHÔNG ĐƯỢC ĐI KÈM COMMENT**.
2. **Xóa comment cũ**: Với các đoạn mã nguồn đã code ra và có comment trong file bạn đang thao tác, hãy **xóa các comment đó**.

Bạn tuyệt đối phải tuân thủ nghiêm ngặt quy tắc này ở tất cả các lần viết và sửa mã nguồn của dự án này.
